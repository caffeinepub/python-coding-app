# Specification

## Summary
**Goal:** Add at least 8 more Python lessons to the backend's pre-populated lesson list, covering a variety of topics and difficulty levels.

**Planned changes:**
- Add at least 8 new lessons to the backend actor's pre-populated lesson list
- New lessons cover Beginner, Intermediate, and Advanced difficulty levels
- Each new lesson includes a title, description, topic emoji, difficulty, and a Python code example

**User-visible outcome:** Users will see a significantly expanded lesson library with more diverse topics and difficulty levels to learn from.
